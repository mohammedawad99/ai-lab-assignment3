# Report evidence snapshots

These are small copies of the final result files that the report cites:

- `cvrp_all_summary.csv` — per instance/algorithm CVRP summary
- `ackley_d10_summary.csv` — per algorithm Ackley summary
- `gp_gep_comparison_runs.csv` — the six GP/GEP comparison runs
- `final_execution_manifest.json` — what ran, with which budgets and times

The full generated outputs (raw per-run CSVs, solution files, extra assets)
stay under `results/final_experiments/` locally and are not committed.
